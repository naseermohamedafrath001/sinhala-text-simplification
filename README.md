# Sinhala Text Simplification Research Project

## üìå Project Overview

This project focuses on **Sinhala Text Simplification** using Natural Language Processing (NLP) and Machine Learning techniques. The main goal of the research is to automatically simplify complex Sinhala text into a more readable and understandable form while preserving the original meaning.

We trained, modified, and evaluated our model using an existing publicly available dataset and further fine-tuned it to suit our research objectives.

---

## üìÇ Dataset Used

We used the **Sinhala Text Simplification Dataset and Evaluation** provided by the BrainSharks FYP17 research group.

### Dataset Source

* **Repository Name:** Sinhala-Text-Simplification-Dataset-and-Evaluation
* **GitHub URL:** [https://github.com/brainsharks-fyp17/Sinhala-Text-Simplification-Dataset-and-Evaluation](https://github.com/brainsharks-fyp17/Sinhala-Text-Simplification-Dataset-and-Evaluation)

### Dataset Description

The dataset consists of parallel Sinhala sentence pairs, including:

* Complex Sinhala sentences
* Corresponding simplified Sinhala sentences

This dataset is designed to support research in:

* Text simplification
* Natural Language Processing (NLP)
* Sinhala language understanding
